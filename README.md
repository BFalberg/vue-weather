# Vue Weather

A personal learning project for exploring Vue 3 concepts — reactivity, composables, Pinia state management, and Vue Router — through a practical weather app. Search for any city to get current conditions and a forecast, and save your favourite cities for quick access.

> **Note:** This is a learning-focused project, not a production application. The primary goal is hands-on practice with Vue's core concepts.

**Live demo:** [weather.benjaminfalberg.dk](https://weather.benjaminfalberg.dk/)

## Features

- Search for cities worldwide
- Current weather conditions and forecast
- Save and manage favourite cities

## Tech Stack

- Vue 3 + TypeScript
- Vite+ (Rolldown-based build toolchain)
- Tailwind CSS v4
- Pinia (state management)
- Vue Router

## Project Setup

```sh
npm install
```

### Development

```sh
npm run dev
```

### Build

```sh
npm run build
```
